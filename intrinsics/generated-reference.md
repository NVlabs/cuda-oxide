## Packed-atomic contracts

- `packed_atomic_add_bf16x2` (`bf16x2`): the native PTX instruction starts at `sm_90`; cuda-oxide admits it from sm_90+. Omitted `.sem` / `.scope` mean relaxed GPU scope. Each 16-bit element rounds to nearest-even and `.noftz` preserves subnormal inputs and results. The elements are atomic independently, so the returned `u32` contains old per-element values that may not form one coherent pair. The pointer must address four writable, four-byte-aligned global bytes; do not mix whole-word or non-atomic overlapping access, and every racing atomic must use a mutually inclusive scope.
